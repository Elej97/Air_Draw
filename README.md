# ✨ El Hareketi ile Havada Çizim (Air Drawing)

Bu proje, webcam üzerinden el hareketlerini algılayarak havada çizim yapmanı sağlar.

MediaPipe ve OpenCV kullanılarak parmak hareketleri gerçek zamanlı takip edilir ve ekrana pürüzsüz çizimler olarak yansıtılır.

🎥 Demo: (buraya Instagram video linkini koy)

---

## 🧠 Özellikler

- ✋ MediaPipe ile el takibi
- 🎨 İşaret parmağı ile havada çizim
- 🖐 El hareketi ile ekran temizleme
- ✌ Çizim / silme modu
- 🎯 Yumuşak çizim (smoothing)
- 🎨 Renk seçme arayüzü
- 🖱 Mouse ile renk seçimi

---

## 🎮 Kontroller

- ☝ İşaret parmağı → Çizim yap
- ✌ İki parmak → Seçim / silme modu
- 🖐 Açık el → Ekranı temizle
- 🎨 Renkleri tıklayarak değiştir
- Q → Çıkış
